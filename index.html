Overview

Aims and Goals

The Lightweight AI-Based Intrusion Detection System (LAIDS) Project aimed to propose a series of lightweight Intrusion Detection Systems (IDSs) to address security vulnerabilities found in resource-constrained networks, particularly within the context of Internet of Things (IoT) and edge devices. The project focused on the application deep learning based architectures in AI-based IDSs.

Proposed Architectures

We proposed and investigated six model architectures:

  • A hybrid Convolutional Neural Network / Gated Recurrent Unit (CNN-GRU) model
  • A hybrid Generative Adversarial Network / Multilayer Perceptron (GAN-MLP) model
  • A 1D CNN model enhanced with Principal Component Analysis (PCA-CNN)
  • A hybrid Autoencoder / Multilayer Perceptron (AE-MLP) model
  • A hybrid Convolutional Neural Network / Long Short-Term Memory (CNN-LSTM) model
  • A Deep Belief Network (DBN) model

Evaluation

These architectures underwent Post-Training Quantisation, specifically testing the precisions of both dynamic and full Float32, Float16, and INT8. The baseline architectures and their quantized variants were evaluated against a baseline 1D CNN. Models were trained and assessed on the CICIDS2017 dataset across three metrics:

  • Multi-classification performance such as accuracy and F1-score
  • Resource requirements including memory footprint, inference latency, and CPU processing time
  • The models’ ability to generalize to detect unknown (zero-day) attacks

Results

From our evaluation, the LAIDS Project demonstrated that lightweight AI models can maintain a high level of security performance. The quantised CNN-LSTM, CNN-GRU, and 2D CNN models were identified as the most promising architectures, offering practical cybersecurity solutions suitable for deployment in resource-constrained network environments.
